## FAQ：买之前最常问的几个问题

### DMIT 49.90 的套餐支持几台设备同时使用？

VPS 本身没有"设备数"限制，你拿到的是一台独立虚拟服务器，怎么用取决于你自己的配置，不存在设备数上限的问题。

### 洛杉矶和香港的 49.90 套餐有什么区别？

两个都是 49.90，但配置差异很大。LAX Pro MICRO 是 2 vCPU / 2GB RAM / 4TB 流量 / 1Gbps 带宽；HKG Pro MINI 是 1 vCPU / 2GB RAM / 500GB 流量 / 300Mbps 带宽。香港节点延迟更低（对大陆用户通常在 30-50ms），但流量和带宽明显更少，价格相同但侧重点不同。

工单系统为主，响应时间通常在几小时以内，紧急问题处理速度在同价位服务商里属于正常水平。不提供实时在线客服。

### 套餐可以中途升级吗？

可以，官网支持套餐升级，差价按剩余时长折算。降级政策建议在购买前直接查看官网条款，不同套餐规则有差异。

### CN2 GIA 和普通 CN2 有什么实际区别？

CN2 GIA 是双向 CN2，去程和回程都走优质线路；普通 CN2（GT）只有去程优化，回程走普通 163 网络。晚高峰差距最明显，GIA 在拥堵时段的稳定性显著更好。
